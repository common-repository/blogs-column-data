A simple plugin to view blog columns, see blog IDs, word counts, and filter blogs by thumbnails.

== Description ==
This plugin lets you view blog columns, see blog IDs, word counts, and filter blogs by thumbnails. It's a handy tool for organizing and managing your blogs.

Restoring a user’s access is simple—just assign them their previous role, and you're all set!

== Installation ==
Installation is straightforward:
1. Install the plugin from the WordPress plugin repository.
2. Alternatively, download the plugin from WordPress.org, then go to your WordPress dashboard.
   - Navigate to Plugins -> Add New -> Upload Plugin -> Choose File -> Install Now -> Activate.
3. You can also search for the plugin in your dashboard under Plugins -> Add New, then install and activate it directly.

== Frequently Asked Questions ==

= Can I view my posts and pages IDs? =
Yes, you can view all post and page IDs.

= Can I see the word count for my blogs? =
Yes, the plugin displays the total word count for each blog.

= Can I filter posts by thumbnails? =
Yes, you can filter posts based on whether they have thumbnails.

= Can I view post visitors? =
No, this feature is not currently available, but we may add it in the future.

= Is there a pro version of this plugin? =
No, we have not yet developed a pro version.
